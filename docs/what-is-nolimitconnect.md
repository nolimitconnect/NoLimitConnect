# What Is NoLimitConnect? (Simple Explanation)

NoLimitConnect is a private communication app where you can run your own network instead of depending on a big company.

If normal social apps feel like renting space in someone else's building, NoLimitConnect is owning your own place.

## Why People Use It

- No account signup, no registration walls
- No ads, no tracking, no data-mining profile
- Encrypted user traffic
- Share large files directly with friends
- Stream your own media from your own device
- Use built-in camera server features (including recording)

## How It Works in 60 Seconds

Think of the network as 2 helper services plus user-hosted rooms.

1. **Network Host**: a public listing board of available hosts.
2. **Connection Test Host**: checks whether your device port is open.
3. **User Hosts**: where people actually meet and share (Group, ChatRoom, Random Connect).

When a user host is enabled, it re-announces itself to the Network Host about every 15 minutes so the listing stays current.

Important: there is no separate "server app" and "client app". It is all in NoLimitConnect.

## Why Host Your Own Network?

Hosting your own network means:

- You control uptime and access rules
- Your community is not tied to one company's policy changes
- You can run private communities for family, clubs, teams, or local groups
- You can still allow public access if you want

## Is Setup Hard?

Usually no. Most people only need to fill a few fields:

- Network Host URL
- Connection Test URL
- Enable UPnP checkbox
- Connection mode choice (test, assume direct, or relay)

If your port is open, you can host directly.
If your port is not open, relay mode still lets you communicate.

## Private vs Public Networks

- **Private network**: best for family, friends, or organizations that want control.
- **Public network**: best for discovery and open community access.

You can start public, then move private later.

## Permission Levels (Simple Version)

Permissions are like access badges:

- 0 Blocked: no access
- 1 Anonymous: very limited
- 2 Guest: basic shared access
- 3 Friend: broader access
- 4 Admin: full control

Permissions are set per person and per plugin, so you can choose exactly who can access camera, media, file sharing, and other features.

## Real-World Use Cases

- Family media hub: stream your home media to your phone
- Private camera feed: monitor a location you control
- Large file delivery: send big files without cloud services
- Community network: run your own group without corporate lock-in

## Bottom Line

NoLimitConnect is for people who want communication tools that they control.

You do not need to be highly technical to start.
Start with the default network, then host your own when you are ready.
