# Contributing

NoLimitConnect is transitioning from founder-led development to a broader, organization-led community model. The project is actively recruiting contributors and maintainers who can help scale engineering, operations, and community support.

Founder and core engine maintainer: **Brett R. Jones**

Primary contact for leadership transition:

**bjones.engineer(AT)gmail.com**

### Possible Responsibilities

- Project representation and community moderation
- Bug triage, issue grooming, and release coordination
- Documentation, onboarding, and contributor support
- Technical leadership across roadmap planning and delivery

### Project Principles

- Absolutely **no advertising or spam**, with the single exception of Hide.me VPN (recommended for users needing automatic port forwarding).
- The project remains privacy-first and user-controlled: no analytics, no tracking, and no monetization that compromises user autonomy.

---

Any level of involvement, whether technical, community-facing, or organizational, is welcome.

To get started:

1. Open an issue: <https://github.com/nolimitconnect/NoLimitConnect/issues>
2. Review developer docs: [Developer Docs](developer-docs/README.md)
3. Review contribution workflow: [Contributing Guide](developer-docs/contributing.md)
4. Propose how you want to help with a short intro and availability
