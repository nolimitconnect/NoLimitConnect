
=== prequisites ===
visual studio 2022 with c/c++ module (Community Edition is fine)
visual studio 2019 is currently supported but will probably be removed in the future

=== for Visual Studio 2022 ===
from visual studio open F:\nolimitapp\nolimitgui\build\VS2022\NoLimitConnectAll.sln
select Release or Debug 
set NoLimitConnect project as startup project
build

=== for Visual Studio 2019 ===
from visual studio open F:\nolimitapp\nolimitgui\build\VS2019\NoLimitConnectAll.sln
select Release or Debug 
set NoLimitConnect project as startup project
build
