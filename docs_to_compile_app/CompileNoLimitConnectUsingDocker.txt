
=== Compile No Limit Connect using docker ===

NOTE: These are instructions for Qt 6.6.2 which is tag nlcdevimage-1.0.7 .. for Qt 6.6.1 use tag nlcdevimage-1.0.5 .. substitute version and tag as needed

1.) Run docker image

docker run -it brettrjonesdocker/nolimitappimage:nlcdevimage-1.0.7 bash
or if dns resolve is broken (recent docker and/or ubuntu changes broke it)
docker run --net=host -it brettrjonesdocker/nolimitappimage:nlcdevimage-1.0.7 bash


2.) Create folder for source code check out

mkdir ~/builds
mkdir ~/builds/nolimitconnectapps
cd ~/builds/nolimitconnectapps

3.) Check out source code

git clone https://gitlab.com/nolimitconnectapps/nolimitapp.git

cd ~/builds/nolimitconnectapps/nolimitapp

4.) Create build folders
either run 
./create_build_directories.sh
cd build/nlc_build

or

mkdir package
mkdir package/linux
mkdir package/android
mkdir build/nlc_build
cd build/nlc_build
rm -rf *


5.) Configure NLC

For linux build:
cmake -S ../.. -B . -DCMAKE_BUILD_TYPE=Release -DTARGET_OS_LINUX=true -DMAKE_ENV=CI  -DQT_VERSION=6_6_2 

For android aarch64 (arm64-v8a) build:
cmake -S ../.. -B . -DCMAKE_BUILD_TYPE=Release -DTARGET_OS_ANDROID=true -DMAKE_ENV=CI  -DQT_VERSION=6_6_2 -DARM64=true

For android arm7a (armeabi-v7a) build:
cmake -S ../.. -B . -DCMAKE_BUILD_TYPE=Release -DTARGET_OS_ANDROID=true -DMAKE_ENV=CI  -DQT_VERSION=6_6_2 -DARM32=true


6.) Build NLC

make -j$(nproc)
